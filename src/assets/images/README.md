# Изображения для Motomarket52

## Локальные изображения

Все фоновые изображения хранятся локально в этой папке:

- `quad-hero-1.jpg` - используется на главной странице и странице "О нас"
- `quad-hero-2.jpg` - используется на странице каталога товаров

## Как добавить новые изображения:

1. Скачайте изображение квадроцикла/мототехники
2. Сохраните в эту папку (`assets/images/`)
3. Используйте в компонентах через путь: `/assets/images/filename.jpg`

## Где используются изображения:

- `home.component.scss` - главная страница (quad-hero-1.jpg)
- `products.component.scss` - каталог товаров (quad-hero-2.jpg)
- `about.component.scss` - страница "О нас" (quad-hero-1.jpg)

## Рекомендуемые размеры:

- **Hero секции**: 1920x1080px или больше
- **Фоновые изображения**: 1920x800px
- **Карточки товаров**: 800x600px
- **Миниатюры**: 400x300px
